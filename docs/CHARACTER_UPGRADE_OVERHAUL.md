# Character Upgrade Overhaul — 3-Path System (9 upgrades per character)

## System Design

**BTD6-inspired 3-path upgrade tree:**
- 3 paths per character: OFFENSE / UTILITY / ULTIMATE
- 3 tiers per path (9 total upgrades)
- Can max ONE path to tier 3
- Can invest up to tier 2 in a SECOND path
- Can only reach tier 1 in the THIRD path
- This creates meaningful build choices: "Do I go full damage or invest in utility?"

**Cost scaling:** T1 = cheap, T2 = moderate, T3 = expensive
- Path A: 120 / 350 / 900
- Path B: 120 / 350 / 900
- Path C: 150 / 400 / 1200

**All ability names reference the SOURCE NOVEL — no copyrighted material.**
